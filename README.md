## Features

- **RAG Integration** - Upload documents (PDF, text) and chat with your knowledge base
- **Smart Chunking** - Optimized document processing with semantic and hybrid chunking
- **Modern UI** - Clean, responsive interface built with Next.js 16 and React 19
- **Voice Mode** - Real-time voice conversations with TTS/STT support4
